Phonelink is a phone number verification SDK for web and mobile apps.

- **Drop-in integration** — a few lines of code to add phone verification to any app
- **Cross-platform** — works on web, React, React Native (Expo), and iOS
- **Secure by default** — cryptographic nonce + signed JWT, validated server-side
- **E.164 output** — get verified phone numbers in a standard, portable format
